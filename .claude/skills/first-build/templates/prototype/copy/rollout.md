# Rollout plan

## Phase 1 — weeks 1–2: pilot with the growth pod

One pod uses it end-to-end. We capture friction in a shared doc and fix the top three weekly.

## Phase 2 — weeks 3–4: expand to platform + mobile

Two more pods join. We measure cycle-time delta against the previous quarter.

## Phase 3 — week 5: company-wide

Default for all teams. Old workflow deprecated. Findings doc becomes a read-only archive.
